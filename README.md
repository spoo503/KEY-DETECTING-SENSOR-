# KEY-DETECTING-SENSOR🔑

🅿️ PROBLEM 
Losing keys is something that happens to almost everyone. Sometimes it takes a lot of time to find them, especially when we are in a hurry.

💡 SOLUTION 
This project is a simple way to solve that problem. It uses an Arduino with a buzzer and LED to help locate keys easily. It can be especially useful for elderly people.

📜 COMPONENTS REQUIRED 
Arduino Uno
Buzzer
LED
Bluetooth Module (optional)
Battery

#📳 Working
The device is attached to the keys.
When it is triggered, the Arduino sends a signal to the buzzer and LED.
The buzzer makes a sound and the LED glows, making it easy to find the keys.
